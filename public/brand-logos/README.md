# Brand logos

Drop the brand logo files here with these exact filenames:

- `bet-bus-logo.png` — BETBUS logo (transparent PNG)
- `eluck-logo.png` — ELUCK logo (transparent PNG)
- `funilive-logo.jpg` — FUNILIVE logo (JPG, cat mascot)

When the user toggles "Incluir logo da marca" ON in Image Studio or
Storyboard, the corresponding logo is composited in the top-right corner
of the generated image client-side via Canvas (no Storage cost).

If the file isn't present, the toggle has no effect for that brand.
